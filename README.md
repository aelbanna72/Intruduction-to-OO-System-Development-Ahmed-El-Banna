# Intruduction-to-OO-System-Development-Ahmed-El-Banna

This project is about creating stock items of different types, adding and selling with error handling, changing stock prices, with a view of the items in a table with VAT calculated automatically. 
There are different classes like StockIteam (base class), NavSys (subclass), CarTyre (subclass), EngineOil (subclass), CarBattery (subclass), TestNavSys and TestPolymorphism (test classes).
